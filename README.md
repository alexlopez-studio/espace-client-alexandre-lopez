# Espace client Alexandre Lopez

Portail client autonome pour le suivi d'un avis de valeur, des documents, des visites, du plan de vente et des offres.

## Développement local

Prérequis : Node.js.

```bash
npm install
npm run dev
```

## Vérifications

```bash
npm run lint
npm run build
```

## Déploiement

Le projet est prévu pour Vercel avec Vite :

- Build command : `npm run build`
- Output directory : `dist`
- Domaine cible : `https://espace.alexandrelopez.fr`
